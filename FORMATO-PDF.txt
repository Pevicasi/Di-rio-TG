FORMATO OBRIGATÓRIO DO PDF — TIRZETRACK v1

O PDF pode ter qualquer aparência nas páginas visíveis do relatório.
No final do documento, deve existir uma seção de texto com exatamente estas três partes:

TIRZETRACK_DATA_START
<JSON COMPLETO CODIFICADO EM BASE64 UTF-8>
TIRZETRACK_DATA_END

Regras:
1. Os marcadores devem ser escritos exatamente como acima, em letras maiúsculas.
2. Entre os marcadores deve haver somente o Base64 do JSON.
3. O Base64 pode ser dividido em várias linhas.
4. schemaVersion deve ser 1.
5. Datas devem usar DD/MM/AAAA.
6. Pesos devem ser números em kg usando ponto no JSON, por exemplo 111.30.
7. Todos os registros anteriores devem continuar no novo PDF; a importação substitui o conjunto inteiro de dados.

IMPORTANTE SOBRE O GITHUB PAGES:
A importação salva os dados no navegador do aparelho usando localStorage.
Ela não modifica os arquivos do repositório nem atualiza automaticamente o site para outros aparelhos.
